GET Select_event_by_date

                        
    https://api.arguss.ai/eventos/2021-09-02

1- URI: https://api.arguss.ai/eventos/

Date format - aaaa-mm-dd

2- Request method - GET

Input parameter specification - header:

                        
    {
        Content/Type: application/json
        Authorization: Bearer { token }
    }

Type- JSON

                        
    {

    "Operacion": "ABI MEXICO_CMM_ZEMOG",
    "Flota": "TZ454_43AN7T",
    "Evento": "Bostezo",
    "Hora_alarma": "2021-08-28 00:00:15",
    "Velocidad": 25,
    "Lat": 19.470199584960938,
    "Long": -99.17579650878906,
    "Photo_url": "https://xxxxxx/",
    "Video_url": "https://xxxxxx/"
    "Id": XYX

    },

Data Specification

Operation: name of the underlying cliente operation;

Flota: plate number of the vehicle;

Hora_alarma: date and time of the occurrence of the alarm;

Velocidad: speed of the vehicle at the alarm moment;

Lat: latitude of the vehicle at the alarm moment;

Long: longitude of the vehicle at the alarm moment;

photo_url: lurl of the photo media;

Video_url: url of the video media;

id: id of the event;

Headers

Content-Type

application/json

Authorization

Bearer

                        
eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJleHAiOjE2MzA2NzY5MjksImlhdCI6MTYzMDY3NjYyOSwibmJmIjoxNjMwNjc2NjI5LCJpZGVudGl0eSI6Mn0.v25ZbaDJ9C1ET4QZP52KZLt3FxwZ7Mex4H-7JRhE77s
